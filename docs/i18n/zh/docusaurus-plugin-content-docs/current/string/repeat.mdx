对字符串进行指定次数的重复连接

### 参数
<ParamTable data={[{"name":"StrSource","type":"string","description":"要重复的字符串"},{"name":"Count","type":"number","description":"要重复的次数"}]} />

### 示例

```ts
import type tt from 'type-toolkit';

type T0 = tt.S.Repeat<'-', 3> // '---'

type T2 = tt.S.Repeat<'-_-', 0> // '-_-'
```
