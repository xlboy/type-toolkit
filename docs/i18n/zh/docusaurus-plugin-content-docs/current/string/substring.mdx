从字符串的指定位置提取子字符串

### 参数
<ParamTable data={[{"name":"StrSource","type":"string","description":"要从中提取的字符串"},{"name":"Start","type":"number","description":"开始提取的位置"},{"name":"End","type":"number","default":"-1","description":"要停止提取的位置，如果为 `-1` 则使用字符串的结尾"}]} />

### 返回值
<ReturnValue data={{"type":"string","description":"提取出的子字符串"}} />

### 示例

```ts
import type tt from 'type-toolkit';

type T0 = tt.S.Substring<'12345', 1, 3> // '23'
type T1 = tt.S.Substring<'12345', 1> // '2345'
type T2 = tt.S.Substring<'12345', 1, 2> // '2'
```
