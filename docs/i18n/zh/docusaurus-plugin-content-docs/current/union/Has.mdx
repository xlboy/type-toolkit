是否包含某个元素

### 参数
<ParamTable data={[{"name":"U","description":"要检查的联合值"},{"name":"T","description":"要检查的目标元素"}]} />

### 返回值
<ReturnValue data={{"type":"boolean","description":"如果 `U` 包含 `T` 则为 `true`，否则为 `false`。"}} />

### 示例

```ts
import type tt from 'type-toolkit';

type T0 = tt.U.Has<'1' | '3', '1'>; // true
type T1 = tt.U.Has<'1' | '3', '2'>; // false
```
