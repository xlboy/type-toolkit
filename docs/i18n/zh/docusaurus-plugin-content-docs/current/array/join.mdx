使用指定的分隔符将数组的元素连接成一个字符串

### 参数
<ParamTable data={[{"name":"Strings","type":"ReadonlyArray<string | number>","description":"要连接的字符串数组"},{"name":"Delimiter","type":"string","description":"用于连接字符串的分隔符","default":"\"\""}]} />

### 返回值
<ReturnValue data={{"type":"string","description":"连接好的字符串"}} />

### 示例

```ts
import type tt from 'type-toolkit';

type T0 = tt.A.Join<['a', 'b', 'c'], '-'> // 'a-b-c' 
type T1 = tt.A.Join<['a', 'b', 'c'], '->'> // 'a->b->c'
type T2 = tt.A.Join<['a', 'b', 'c'], ' or '> // 'a or b or c'
```
