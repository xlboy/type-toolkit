Create an array of numbers progressing from the given start value up to, but not including the end value.

### Parameters
<ParamTable data={[{"name":"Start","type":"number","description":"The start of the range."},{"name":"End","type":"number","description":"The end of the range."}]} />

### Return Value
<ReturnValue data={{"type":"number[]","description":"The range of numbers."}} />

### Example

```ts
import type tt from 'type-toolkit';

type Range1 = tt.N.Range<1, 5>; // [1, 2, 3, 4]
type Range2 = tt.N.Range<1, 1>; // []
type Range3 = tt.N.Range<55, 60>; // [55, 56, 57, 58, 59]
```
