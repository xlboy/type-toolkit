Repeat a string a specified number of times

### Parameters
<ParamTable data={[{"name":"StrSource","type":"string","description":"The string to repeat"},{"name":"Count","type":"number","description":"The number of times to repeat the string"}]} />

### Return Value
<ReturnValue data={{"type":"string","description":"The repeated string"}} />

### Example

```ts
import type tt from 'type-toolkit';

type T0 = tt.S.Repeat<'-', 3> // '---'
type T1 = tt.S.Repeat<'-_-', 3> // '-_--_--_-'
type T2 = tt.S.Repeat<'-_-', 0> // '-_-'
```
