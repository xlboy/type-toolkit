使用指定的分隔符将数组的元素连接成一个字符串

### Parameters
<ParamTable data={[{"name":"Strings","type":"ReadonlyArray<string | number>","description":"The string array to join"},{"name":"Delimiter","type":"string","description":"The delimiter used to join the strings","default":"\"\""}]} />

### Return Value
<ReturnValue data={{"type":"string","description":"The joined string"}} />

### Example

```ts
import type tt from 'type-toolkit';

type T0 = tt.A.Join<['a', 'b', 'c'], '-'> // 'a-b-c' 
type T1 = tt.A.Join<['a', 'b', 'c'], '->'> // 'a->b->c'
type T2 = tt.A.Join<['a', 'b', 'c'], ' or '> // 'a or b or c'
```
