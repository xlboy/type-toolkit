Adds all the elements of an array into a string, separated by the specified separator string

### Parameters
<ParamTable data={[{"name":"Array","type":"ReadonlyArray<string | number>","description":"The array to concat"},{"name":"Separator","type":"string","description":"A string separator","default":"\"\""}]} />

### Return Value
<ReturnValue data={{"type":"string","description":"The concatenated string"}} />

### Example

```ts
import type tt from 'type-toolkit';

type T0 = tt.A.Join<['a', 'b', 'c'], '-'> // 'a-b-c' 
type T1 = tt.A.Join<['a', 'b', 'c'], '->'> // 'a->b->c'
type T2 = tt.A.Join<['a', 'b', 'c'], ' or '> // 'a or b or c'
```
