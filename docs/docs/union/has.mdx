Check if an element is included in a union type

### Parameters
<ParamTable data={[{"name":"U","description":"The union value to check"},{"name":"T","description":"The target element to check"}]} />

### Return Value
<ReturnValue data={{"type":"boolean","description":"`true` if `T` is included in `U`, otherwise `false`."}} />

### Example

```ts
import type tt from 'type-toolkit';

type T0 = tt.U.Has<'1' | '3', '1'>; // true
type T1 = tt.U.Has<'1' | '3', '2'>; // false
```
